# 需求管理
1. 在文件目录中，可创建文件夹用
2. 文件夹下创建需求文件
3. 文件夹支持编号，可创建多个文件将需求归类

![需求目录树](./images/requirement.png)

设置文件编号后可通过编号查看文件
![需求文件](./images/requirement-info.png)

可在一个目录中将大的需求文件拆分多个小文件，多个人一同编写。
不同的人员快速定位到自己需要关心的部分，适合多人分工协同编写。
