# 为什么选择我们
1. 从前端到后端，从AI到开发，测试，产品支持一体化协同。
2. 前端/后端 可支持多种语言代码的生成。
3. 完善的自动化机制，自动生成相关测试用例。
4. 模型驱动，自动生成并适配契约，前端无需关系后端接口。
5. 极简的开发，前端无需关系契约和适配接口，后端只需要写一个函数。
6. 自动适配，前端界面修改，后端接口修改可自动适配，无需手动修改。
7. 极快的业务开发速度，后端一个函数一个功能，前端页面设计器可直接在线修改实时生成生产级别的代码。

# 适用人群
1. SE 提供需求文件，可直接生成后端代码，前端代码配置自动化测试框架，能够达到生产级别。
2. 测试 在测试模块可输入测试用例，快速完成测试工作。
3. 后端开发 只需要写一个函数即实现了相关的业务功能
4. 前端开发 只需要直接在页面设计器上配置调用接口。
5. 产品经理/UI设计 直接在页面设计器上配置即可生成前端代码
6. AI+协同 AI和用户可协同开发，相互修改内容完全一样没有任何区别。
